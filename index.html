ArkScript

A small, fast, functional and scripting language for C++ projects

A language running on a virtual machine, compiled to home made bytecode to run everywhere.

It's also a functional language, inspired by Lisp and Python, with a very small set of keywords to promote functionalities and expressiveness. You don't need 50 keywords and another 20 for very specific use cases!

It's easy to compile: ark -c script.ark and easy to run: ark script.arkc (ark script.ark also works!)
